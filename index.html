Replacing music or character art

This project uses Web Audio API beeps and CSS-only characters. To replace the placeholder audio, add your own original audio file and hook it into startSongAudio() in game.js.

To replace character art, swap the CSS character divs in the gameplay stage with your own original <img> or <svg> art. Keep the #player and #opponent IDs so the animation hooks still work.
